.mpr ファイルとは何か — mxcli の役割What is .mpr — and what does mxcli do?

.mpr = Mendix プロジェクトファイル.mpr = Mendix Project File

Mendix 独自形式のプロジェクトファイル。
エンティティ・マイクロフロー・ページ・セキュリティ設定がすべてこの1ファイルに格納されている。
Studio Pro はその GUI エディタ。mxcli はコードから直接操作するツール。 Mendix's proprietary project file format.
Entities, microflows, pages, and security rules are all stored in this single file.
Studio Pro is the GUI editor. mxcli manipulates it directly from code.

MDL — Mendix Definition Language

mxcli が解釈する SQL ライクな DSL。
Mendix のすべての要素を宣言的に記述できる。 A SQL-like DSL interpreted by mxcli.
Describe any Mendix element declaratively. 

CREATE ENTITY → ALTER PAGE → GRANT EXECUTE

⚠️ Studio Pro でプロジェクトを開いている間は mxcli で編集しないこと。
Studio Pro はメモリキャッシュを持ち、外部からの変更を認識できない。
最悪 .mpr が破損する。
手順: Studio Pro でプロジェクトを閉じる → mxcli 実行 → Studio Pro で再度開く ⚠️ Do NOT edit with mxcli while the project is open in Studio Pro.
Studio Pro holds in-memory caches that cannot be updated externally — concurrent edits can corrupt the .mpr file.
Workflow: Close project in Studio Pro → run mxcli → re-open in Studio Pro

DESCRIBE — ビジネスロジックを読むDESCRIBE — Reading Business Logic

$ ./mxcli -p Helpdesk2.mpr -c "DESCRIBE MICROFLOW HD.ACT_Ticket_Submit"

BEGIN
  IF $Ticket/Priority = HD.TicketPriority.Critical THEN
        CHANGE $Ticket (Status = Open, SLADueAt = addHours([%CurrentDateTime%], @HD.SLA_CRITICAL_HOURS));
  ELSE IF $Ticket/Priority = HD.TicketPriority.High THEN
    CHANGE $Ticket (Status = Open, SLADueAt = addHours([%CurrentDateTime%], @HD.SLA_HIGH_HOURS));
  ELSE
    CHANGE $Ticket (Status = Open, SLADueAt = addHours([%CurrentDateTime%], 24));
  END IF;
  COMMIT $Ticket;
  RETURN true;
END;

SHOW CONSTANTS IN HD で値を確認できる（= 2時間） 📌 @HD.SLA_CRITICAL_HOURS is a constant ref.
SHOW CONSTANTS IN HD shows its value (= 2 hrs)

# mdlsource/ex-category.mdl
CREATE ENUMERATION HD."TicketCategory" (
  "Hardware" CAPTION 'ハードウェア',
  "Software" CAPTION 'ソフトウェア',
  "Network"  CAPTION 'ネットワーク',
  "Other"    CAPTION 'その他'
);

ALTER ENTITY HD."Ticket"
  ADD "Category": Enum HD.TicketCategory;

ALTER PAGE HD.Ticket_NewEdit {
  INSERT AFTER txtSubject {
    COMBOBOX cbCategory (
      Label:     'カテゴリ',
      Attribute: Category
    )
  }
}
./mxcli check ex-category.mdl -p Helpdesk2.mpr --references
./mxcli exec  ex-category.mdl -p Helpdesk2.mpr
